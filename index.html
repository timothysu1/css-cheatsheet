Flexbox Row

Use these three properties to create a Flexbox row layout.

        
          .row {
            display:flex;
            flex-direction: row;
            flex-wrap: wrap;
          }

Flexbpx Column

Use this to create a flexbox column layout.

        
          .column {
            display: flex;
            flex-direction: column;
          }

CSS Grid Layout

Build a 12-column layout using CSS Grid

        
          .grid {
            display: grid;
            width: 100%;
            grid-template-columns:
          repeat(12, 1fr);
          }

Linear Gradients

This will create a background linear gradient from the top to bottom.

        
          .linear-gradient-background {

          gradient(
              rgba(232, 102, 236, 0.3) 0%,
              rgba(232, 102, 236, 0.6) 100%
            );
